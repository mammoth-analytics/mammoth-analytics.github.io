### Product Enhancement :

#### Smart Column Placement for SQL Query Rule –
Enhanced the SQL Query rule to place result columns adjacent to their source columns in the grid, rather than appending them to the end.

#### Pipeline UI Refinements –
Refined the Pipeline UI by replacing the fade effect with a shadow on the plus button area, and reverting the Draft Mode indicator to its previous style with consistent coloring.

### Defect Fixes :
Resolved the following issues:

* On the Plans page, hourly pulls for Business Plan orchestration (self-serve) were incorrectly listed.
* View Summary failed to load from the View menu on the app.
* In Export to DB, the "Keep in the end of pipeline" setting was being unexpectedly changed.
* In Dataflow Monitor, the Active Tasks tab displayed a blank state for tasks with Auto Sync off, despite tasks being present.
* In Dataflow Monitor, long dataset names caused overlapping text over the view name in the Active Tasks tab.
* Fill Missing Values – A backward compatibility issue caused existing rules to go into error after a new batch was added.
* BigQuery Export was not completing successfully.
* In Third-party Connectors, saving as a dataset during import with specific queries was not working.
