### Product Enhancement :

#### Profile Settings Enhancement–

IIntroduced a new UI for Profile Settings, providing an improved and more intuitive user experience.


### Defect Fixes :
Resolved the following issues:

* SFTP data import showed incorrect folder counts when search was applied.
* In the Data Library, the search bar caused the vertical scrollbar to automatically reset to the top.
* Drag and drop files into the Data Library did not open the New Dataset modal.
* In self-serve plan upgrade, hovering over the back button displayed incorrect text.
* BigQuery import failed when datasets contained JSON type columns.
* In the Preview Panel under Schema Settings, column rename did not take effect when rules or metrics were applied in a view.
* During creation of third-party identity, users were redirected to the file upload tab instead of the connectors tab after completing the OAuth process.
* In workspace settings, inviting a user showed an "Undefined" text in the modal.
* Creating a dataset using AI with the "Let AI Decide" option resulted in an error.
* Live link was not working on the application.
* In User Management, project permission changes did not reflect immediately after updating roles.
