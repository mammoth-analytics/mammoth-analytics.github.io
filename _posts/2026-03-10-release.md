### Product Enhancement :

#### Background Task Execution for Pipeline Operations –

Introduced background execution for pipeline tasks, allowing transformations to run without blocking the interface and enabling continued pipeline development while tasks are processing.

#### SFTP Directory Listing Optimization –

Optimized SFTP file and directory listing to significantly improve performance when browsing directories containing a large number of nested files.

#### SFTP Connection Configuration Enhancement –

Enhanced SFTP connector configuration to support customer-provided private keys and usernames during connection setup.

#### Project Date Format Default Setting –

Updated project settings so that Define date format for this project is unchecked by default, allowing the system to automatically use the user's local date format settings.

#### AI Generate Button Repositioning –

Repositioned the Generate via AI button to improve visibility and accessibility.

#### Large View Duplication Handling –

Moved View Duplication and Query Rows operations to asynchronous processing to prevent interface blocking when working with very large datasets.

#### Preview Panel UI Improvements –

Improved the Preview Panel interface with enhancements to layout, controls, and overall usability.

#### XML File Upload Support –

Added support for uploading and processing XML files within the platform.



### Defect Fixes :
Resolved the following issues:

* Hovering over the Column Browser and Dataset Settings icons in the Data Editor did not display the corresponding name/details.
* In Convert Column tasks, the task description incorrectly showed Format Output = Undefined.
* Data update toggle in the Dataflow Monitor required a page refresh to reflect the updated state.
* Pipeline flickering occurred for large datasets when Draft Mode was enabled.
* Preview Data settings failed when Skip Row changes were applied.
* In Connectors → New Connection, field ordering was inconsistent and the modal closed on validation errors (e.g., incorrect password) instead of displaying the error message.
* In SFTP connections using SSH, the Download Key button and related instructions were missing from the modal.
* Continuous loader appeared after uploading the JSON configuration for Google Cloud Storage.
* Group Pivot Rule execution remained in buffering state.
* Dataset Settings page showed an infinite loader on refresh.
* Pipeline tooltip displayed incorrect information in Draft Mode.
* Unable to create Google Drive datasets.
* Unable to create BigQuery datasets in the application.
* Loader was not displayed when navigating to the root directory from the landing page.
* Uploading an unsupported file type through URL upload did not display an error message.
