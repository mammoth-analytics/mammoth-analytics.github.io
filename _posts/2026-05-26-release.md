### Product Enhancement :

#### Dataset Preview Tables –

Toggle an optional, resizable preview pane when selecting datasets for joins, lookups, and other operations. Expand the preview modal to inspect more data before confirming your selection.

#### Tableau Export –

Export your data and analysis directly to Tableau. Streamline your reporting workflow by pushing insights to your preferred analytics platform.

#### Join Interface Improvements –

Pre-selected join conditions and expanded default fields make configuring joins faster. Set up complex multi-table analyses with less manual configuration.

#### SQL Query Transparency –

Pipeline step cards now display the Intent and generated SQL query for AI/SQL-based features. Understand the natural-language description of what each step does and inspect the underlying SQL logic.

#### Conditional Highlights –

Create custom highlighting rules to draw attention to specific data patterns. Define conditions to highlight cells matching your criteria, making important values and trends instantly visible.

#### Multiple LLM Provider Support –

Add API keys for multiple LLM providers (OpenAI, Anthropic Claude, Grok, Google Gemini) to use with GenAI and other platform features. Choose the best model for your use case and requirements.

### Defect Fixes :

Resolved the following issues:

* Dashboard pipeline cards now display correctly without distortion.
* Resource finder correctly displays selected text and auto-selects when only one matching view exists.
* File collection orchestration no longer throws an error when removing a source from a two-source configuration.
* Workspace modal now displays the workspace name when leaving a workspace.
* Task descriptions now maintain proper spacing when blank lines are inserted.
* Column type conversion in the column browser correctly reflects the converted type in the function panel.
* Metric condition layout no longer appears distorted in backward compatibility mode.
* Column browser scroll bar now scrolls to show all columns in the list.
* Pipeline scroll bar position is retained when minimizing and maximizing.
* Explore card updates correctly after applying a column conversion rule.
* BigQuery export no longer fails due to race conditions during concurrent operations.
* Extract JSON function now operates without errors in the pipeline.
* JSON files display the Extract JSON shortcut in the column menu.
* Data grid sort function now works correctly from the sort footer.
* Branchout draft mode no longer incorrectly treats newly added columns as hidden.
