### Product Enhancement :

#### Kafka Import –

Mammoth now supports Apache Kafka as a data source, letting users pull messages from Kafka topics directly into datasets for analysis. Connections support PLAINTEXT and SASL_SSL authentication protocols.

#### MQTT Import –

Users can now ingest IoT sensor data from MQTT brokers by subscribing to topics and collecting streaming messages into Mammoth datasets. This connector is designed for manufacturing and industrial environments.

#### REST API Export –

Users can now push dataset data to any external REST API endpoint using POST, PUT, or PATCH requests, completing two-way integration alongside the existing REST API import connector.

#### In-App Documentation Links –

Help documentation links are now available directly within several product features, including AI SQL Query, Generative AI, Data Check, Checkpoint, Conditional Highlights, Metrics, Parameters & Snippets, and export destinations.

### Defect Fixes :

Resolved the following issues:

* Data Check — The "Notify on Failure" setting now saves correctly when the toggle is turned off and the task is re-edited.
* Data Library — Dataset column and row counts now display in a human-readable format (e.g., 1.2M) instead of raw unformatted numbers.
* Dropbox Connector — Users can now select files and folders when setting up a Dropbox connection; the selector was previously unresponsive.
* Self-Serve Workspaces — User name and profile avatar now display correctly and are no longer clipped.
* Condition Filter — Builder mode with AI intent no longer throws an error when generating filter conditions.
* User Management — A non-functional checkbox has been removed from the manage users modal.
* Self-Serve Billing — Downgrading a workspace plan from Business to Lite now completes without error.
* Dataview / Explore — Dataview and explore cards no longer show a continuous loading indicator after data loads.
* Data Preview — An error message that incorrectly appeared while fetching data for preview no longer shows in unrelated views.
* Tooltips — Tooltip alignment is now correct across the application.
* Funciton panel - Minor UI improvements to the "Place results into" section for a more consistent experience.
