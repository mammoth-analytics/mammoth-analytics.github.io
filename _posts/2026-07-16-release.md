### Product Enhancement :

#### Dataset Settings –

The dataset settings modal now saves typed changes automatically, without requiring users to press Enter, and its confirmation button is now labeled "Close" for clarity.

### Defect Fixes :

Resolved the following issues:

* Column browser — Column numbers now align correctly after applying an extract JSON rule.
* Extract Date column rule — Changing the extracted date part no longer triggers an invalid column type error that disabled the apply button.
* Manage Storage settings — Dataset lists now load without the extended delay seen previously.
* Manage Storage settings — Deleting a dataset now shows a clear message when it fails, instead of an unexplained error or no response at all.
* Dashboard search — Searching by a dashboard's name or keywords now returns the matching results.
* Activity Log — The Add Data event no longer shows "undefined" in its details.

### Security :

* Addressed a security vulnerability in a third-party JavaScript library, with no impact on existing features or workflows.
