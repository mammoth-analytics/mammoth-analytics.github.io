### Product Enhancement :

#### Download Dashboard as PDF –
Introduced the ability to download dashboards as a PDF directly from the Dashboard view via a download icon, generating a multi-page file with each tab represented as a separate page.

#### Parameters & Snippets –
Introduced project-level Parameters & Snippets accessible from the left navigation, enabling centralized management and reuse of dynamic values and logic across pipelines.

#### Enhanced Storage visibility –
Plan & Storage- Enhanced storage visibility in settings by including detailed dataset size information alongside views for better storage tracking and management.

#### SharePoint Data Import –
Introduced SharePoint as a new import source, allowing users to browse folder structures, select one or multiple files, and retrieve the latest data for imported datasets.

#### Auto Sync Terminology Update –
Replaced Data Update terminology with Auto Sync across the application, with updated states to clearly indicate on (Auto Sync) and off modes.

#### Pipeline UI Redesign –
Introduced a minimized pipeline view and a redesigned function picker with search and categorized transformations, providing a more focused workspace and faster access to pipeline operations.

#### Dataflow Monitor Enhancement –
Enhanced the Dataflow Monitor modal to support project switching via a dropdown, enabling users to view and manage dataflows across different projects more easily.

#### Streamlined Dataflow Monitoring –
Split the unified Dataflow Monitor modal into two distinct modals — Dataflow Monitor for tracking active pipelines, tasks, and orchestrations, and Pending Items for managing pending pipelines, checkpoints, and data checks — providing clearer separation of monitoring and action workflows.

#### Smart Column Name Formatting –
Extended the Schema modal with additional column name formatting options including snake_case, camelCase, and an AI-powered Describe mode, allowing users to auto-generate or standardize column naming conventions across datasets directly from the Dataset Settings.

#### Guided Export Workflow –
Introduced a guided informational step in the Export flow that allows users to review and configure export destination options — including Mammoth managed datasets and external destinations such as Google BigQuery, PostgreSQL, MySQL, Live Link, and Send to Dataset — before applying the export action.

### Pipeline Checkpoints –
Introduced a new Checkpoint step type in transformation pipelines, supporting two modes — Alert (notifies recipients via email or webhook and continues) and Approval Gate (pauses the pipeline until approved) — with pending checkpoints surfaced across the Pending Items modal and Dataflow Monitor, allowing recipients to review and approve directly without entering the pipeline.

### Functions Panel Improvements –
Improved the Functions panel with added tooltips on function names and removed internal card scroll for a cleaner configuration experience.


### Defect Fixes :
Resolved the following issues:

* Renaming a project did not work from the Data Library and Settings page.
* Reopening the Lookup rule after deleting views resulted in an error.
* The horizontal divider between the bottom view tabs bar and the function panel was missing.
* Opening the Function Panel did not show a loader, causing a delay before content appeared.
* In SFTP export, the Apply button was disabled for empty directory names.
* In Activity Log,add data event details did not correctly specify combine or replace actions.
* In Paste from Clipboard, issues with data inclusion, loading state, and validation were resolved.
