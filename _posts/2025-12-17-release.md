### Product Enhancement :

* Intent-Based Display Properties – Enhanced the Data Editor experience by dynamically adjusting column visibility, sorting, and actions based on user intent, making data interactions more intuitive and context-aware.

* Generate Dataset Using AI – Added the ability to generate new datasets using natural language prompts, allowing users to quickly create structured, ready-to-use data directly within the platform.


### Defect Fixes :
Resolved the following issues:

* Preview Panel – Batch Section: Improved row count alignment for better visual consistency.
* Group & Pivot: Fixed an issue where a duplicate column error was shown when copy-pasting a Group & Pivot task in specific scenarios.
* Preview Panel: Fixed an issue where editing a query from the preview panel did not work as expected.
* SFTP Datasets: Resolved an issue where the Change Data Pull Type option was missing for SFTP-based datasets.
* Data Editor: Fixed an issue where the grid went blank when sorting was applied while the pipeline was in an error state.
* Data Editor: Resolved an issue where copy-pasting a transformation in an error state did not display the missing column name when pasted into a different view.
