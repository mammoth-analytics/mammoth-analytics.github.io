### Product Enhancement :

#### Project Settings Modal Enhancement – 
Refined the Project Settings modal with improved layout, streamlined user access controls, enhanced search and role management experience, and better alignment between project creation and editing workflows.

#### Project Access Control Improvements – 
Enhanced project access options with clearer visibility settings for “Only me,” “Everyone in this workspace,” and selective user access, ensuring more intuitive and consistent project-level permission management.

#### Email Notifications for Critical Failures – 
Added email notifications for key failure scenarios across pipelines, data actions, imports, exports, and data updates to improve visibility and timely response.


### Defect Fixes :
Resolved the following issues:

* API keys were fully visible in Settings instead of being securely masked.
* An error toast was displayed when updating Project Settings.
* Dataset generation using AI failed.
* Task description displayed as “undefined” during export to Elasticsearch.
* Full column name was not visible on hover in Explore Cards when aggregate functions were applied.
* Format changes to numeric and date columns were not retained after copying to a new column.
* Date values were truncated in Explore Cards.
* Transform using AI prompt - A toast message was incorrectly triggered for pipeline changes.
* Date format setting in the Preview Panel did not pre-select the Project Settings option as expected.
* Settings-Profile - Uploading a file for the profile picture triggered an error toast.
