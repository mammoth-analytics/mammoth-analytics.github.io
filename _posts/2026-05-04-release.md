## Product Enhancements :

**SQL and Natural Language Expressions in Math, Metric, and Filter** –
Math, Metric, and conditional Filter steps now accept SQL expressions directly, with an AI-assisted mode that generates expressions from a plain-language description. Users can choose whichever input method fits their workflow.

**Data Check: Custom Expression Type** –
Data Check now includes a custom expression type, letting users define validation rules as AI-generated expressions from a plain-language description.

**New SFTP Import Flow** –
A redesigned SFTP import experience makes it easier to connect, configure, and refresh data from SFTP-hosted files.

**OneDrive Export** –
Users can now export datasets directly to Microsoft OneDrive from the export destinations panel.

**Dropbox Folder-Level Access** –
Dropbox connections now support folder-level permissions, letting users restrict access to a specific folder rather than the entire drive.

**Workspace-Level Permissions and Invite Redesign** –
The invite users experience has a refreshed interface and now supports workspace-level permission settings for more granular control over member access.

**Resource Finder Improvements** –
The Resource Finder now opens with the search bar expanded and auto-focused for faster dataset lookup, and automatically navigates to the current dataset's folder when launched from Join or Lookup steps.

**Parameters & Snippets Improvements** –
The Parameters & Snippets panel includes updated interface controls and usability refinements that make managing reusable values more efficient.

**Schema Modal: Column Deletion Protection** –
The schema settings modal now prevents users from removing all existing columns — at least one column must remain, or a new column must be added, before changes can be saved.

**Transform Menu Redesign** –
The transform menu has a refreshed layout that makes browsing and selecting pipeline steps faster and more intuitive.

**Updated Application Font** –
The application interface now uses Inter throughout, replacing the previous typeface for improved visual consistency and readability.

## Defect Fixes :

Resolved the following issues:

* Data Check — Data checks placed as the last step in a pipeline now execute correctly.
* Data Check — Unique value checks now correctly identify non-unique data instead of passing incorrectly.
* Data Check — Reordering a task above a data check step no longer triggers an infinite loading state.
* Data Check — Pipelines with a data check at the last step no longer show a continuous loader after an app update.
* Data Check — A queued data check no longer shows as passed when a checkpoint is awaiting approval upstream.
* Data Check — Parameters now work correctly as values within data check conditions.
* Data Check — Failed data check email notifications are now delivered as expected.
* Pipeline — A clear error message now appears when a user attempts to add a branch out while a failed data check is blocking the pipeline.
* Data Library — Cloned datasets now appear on the landing page immediately after the cloning confirmation toast.
* Data Library — Landing page cards no longer flicker while content is loading.
* Data Library — Deleting a dataset now also removes its associated scheduled run.
* Google Drive — The folder picker now displays the correct folder hierarchy instead of showing all subfolders at the root level.
* Conditional Filter — The IS condition now behaves correctly when applied to a subset of values in intent-based filters.
* Explore — Filter values in Explore cards now display correctly.
* Explore — Unique numeric value search now works without requiring a comma separator.
* User Management — The team members table no longer appears with a distorted layout.
* Self-Serve — A notification now appears when a workspace cannot be downgraded to the Lite plan due to existing members.
* Self-Serve — Users are no longer unexpectedly redirected to the plan selection page.
* App — The maintenance modal no longer appears unexpectedly after logging in.
* App — Form labels no longer wrap to a new line following the font update, restoring the intended layout across configuration panels.
* Pipeline — The Convert Column Type step now displays source and destination data type names with consistent capitalization.
* Dropbox — The import flow no longer prompts users to reselect files after individual files have already been chosen.
* Profile — Profile settings now correctly reflect two-factor authentication status immediately after enabling it.
* Parameters — Parameters used in branch out conditions now apply correctly.
