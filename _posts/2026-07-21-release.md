### Defect Fixes :

Resolved the following issues:

* Draft mode pipelines — Suspending or restoring a branch-out or pivot step now correctly shows the Apply and Discard buttons, matching other rule types.
* Draft mode — The confirmation message now states whether draft mode was turned on or off, instead of a generic status update.
* Monitor — The Fix Settings button in the Needs Attention tab now opens the relevant settings instead of doing nothing.
* File upload — The loading and success icons in the upload progress widget are now properly aligned.
* Monitor — The attention count shown on the Monitor badge now matches the number of items listed in the Needs Attention view.
* Bulk Replace — The new column now appears next to the original column and is selected by default.
* Join — Warnings and errors on a Join step now appear at the top of the card, with an icon and tooltip visible even when details are collapsed.
* Resource Finder — Removed an unnecessary horizontal scrollbar that appeared even when there was enough space to show all content.
* Explore Cards — Selecting multiple values with Shift or Ctrl in list view now works correctly.
* Workspace access — Users with no shared dashboards can now log out or return to the landing page instead of getting stuck.
* AI settings — Corrected spacing around the link to add an API key and nearby text for better readability.
* Join — The Join preview no longer goes blank when a common key column is selected.
