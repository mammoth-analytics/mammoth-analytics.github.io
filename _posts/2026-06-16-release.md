### Product Enhancement :

#### Versioned Pipelines –

Users can now view and restore the full version history of any pipeline in a dedicated version history modal. Every modification automatically creates a new version with timestamps and task-level details, and users can restore any prior version or copy tasks from it directly.

#### Copy and Paste Pipeline Tasks –

Users can now copy and paste one or more pipeline tasks within the same view or into a different view's pipeline, including tasks copied directly from the version history. Pasted tasks enter draft mode automatically, and each pasted step is visually labelled for easy identification.

#### Monitor Tab –

The left navigation now includes a consolidated Monitor tab, bringing the data-flow monitor, pending items, and activity log together in a single location.

#### Column Drag Highlight –

Columns now display a visual highlight when being dragged, giving users a clearer indication of which column is being repositioned in the grid.

### Defect Fixes :

Resolved the following issues:

* Function Panel — The panel's state and configuration now persist correctly after navigating away and returning to the pipeline.
* Bulk Replace — Suggestion text in the bulk replace panel no longer overlaps with surrounding UI elements.
* Send to Dataset — The "Send to Dataset" action now correctly defaults to placing output at the end of the pipeline.
* Branch Out — Editing a mid-pipeline "Save as Dataset" step no longer allows mapping columns that were added after that step.
