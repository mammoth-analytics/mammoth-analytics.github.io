### Product Enhancement :

#### Snippets Improvements –

Snippets in rules now include a one-click button to copy a snippet's value and a dedicated button to insert it directly into a formula. Selecting a snippet type is simpler with the separate SQL and expression tabs removed, and hovering over a snippet in use now shows where it's applied.

#### New Data Page –

The new Data page brings your datasets, automations, and parameters together in one place, making it easier to find and manage everything in a workspace without switching between sections.

---

### Defect Fixes :

Resolved the following issues:

* Workspace settings — Searching for a data source name in the storage management section now returns matching results instead of coming up empty.
* Bulk Replace — Renaming a group repeatedly no longer leaves outdated names showing in the column list.
* Pipeline checkpoints — An approved checkpoint no longer reverts back to a pending state.
* Pipeline checkpoints — Notification emails now include the workspace, project, and data source name instead of just the view name.
* Navigation — Moving from Settings to the Data page no longer redirects to the home page unexpectedly.
* Metrics — Metric values now refresh correctly when a conditional filter rule is applied, instead of showing stale numbers.
