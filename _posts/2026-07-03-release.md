### Product Enhancement :

#### Export Update Method Visibility –

Crosstab and BigQuery exports now display their update method, giving users the same visibility into refresh behavior already available for other export types.

#### Monitor Governance –

The Monitor governance tab now links directly to the view where a pending checkpoint approval is needed, so users can jump straight to it instead of searching manually.

#### Checkpoints –

Checkpoints now use a redesigned interface, making it easier to review and approve pipeline steps.

#### Data Ingestion Date Handling –

Date formats are now recognized and standardized more reliably during data ingestion, and the pending batches view now supports tracking multiple batches at once.

### Defect Fixes :

Resolved the following issues:

* Pipelines — Steps created from AI-generated SQL queries now show their description instead of appearing blank.
* File uploads — The fix-settings option for password-protected files now applies the selected settings correctly.
* Pipelines — Actions no longer change order when copied and pasted between pipelines.
* Lookup rules — The rule description no longer appears blank when the referenced view has been deleted.
* Lookup rules — Editing a rule to point to a different view no longer triggers an invalid destination column error.
* AI rules — Rules configured with a Gemini key no longer fail with a runtime error.
