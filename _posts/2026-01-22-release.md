### Product Enhancement :

* Enhanced error handling in intent-based task generation – Users now get more informative messages when an AI-generated task can’t be processed, making it easier to understand why an action didn’t go through.
* Enhanced Convert Column Type rule – Added flexible date output format options with improved source format selection for more accurate and controlled date conversions.
* Preserved preview panel state across pages – Maintains the selected item in the preview panel when navigating across the Data Library, Dashboards landing, and Orchestrations landing pages.

### Defect Fixes :
Resolved the following issues:

* Fixed issues with date-type columns in the Rows section of Group & Pivot.
* Addressed layout inconsistencies in the Data Quality view to improve overall clarity and usability.
* Resolved multiple issues in the Convert Column Type rule, including incorrect format auto-detection, missing details in task descriptions, and configuration not appearing correctly while editing.
