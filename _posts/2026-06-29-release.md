### Product Enhancement :

#### Trash –

Users can now move views and datasets to Trash instead of permanently deleting them, making accidental deletions reversible. Items can be restored individually or in bulk by folder from a dedicated Trash page.

---

### Defect Fixes :

Resolved the following issues:

* File upload — Tab-delimited and pipe-delimited files with unstructured rows now correctly detect the file delimiter, enable the submit button, and import all rows without skipping any.
* BigQuery export — The export modal now shows a connection selector when multiple identities are available, loading the project list from the correct identity.
* Pipeline — The Branch Out action now displays the update method (replace or combine) in its task description, and copy-pasted Branch Out actions allow changing the update method as expected.
* Pipeline — Copying and pasting a Lookup task no longer creates duplicate column IDs in the pipeline.
* Pipeline — When a suspended rule causes a dependent task error, the function panel now correctly disables the Apply button and highlights the affected column.
* Parameters — All breadcrumb items in the Parameters preview panel (project, folder, and view) now navigate correctly when clicked.
* Settings > Projects — User avatar tooltips now display the username immediately on hover anywhere over the avatar.
* Data connections — Cloud data pulls that fail now send a notification email to alert users to the error.
* Window Functions — Result columns now place adjacent to the grouped column, Count aggregation results no longer show decimal values, and task descriptions now use the correct labels to match the function panel.
* Resource finder — The resizable panel no longer disappears on smaller screen sizes.
* Monitor and Trash — Vertical scrollbars now appear correctly when many projects are listed, making it possible to scroll through the full project list.
