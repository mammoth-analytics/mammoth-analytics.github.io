## Product Enhancements :

#### Transform Intent to SQL –

Transform natural language intents directly into executable SQL expressions for faster query generation.

#### Transform Menu Tab Support –

Tab navigation now works across all sections of the new transform menu for faster keyboard-driven workflows.

#### User Management Enhancements –

Invited user UI has been redesigned for a clearer invitation and user management experience.

#### Functions Panel Export –

Export action layout in the functions panel has been refined with improved padding and alignment.

#### Azure Blob Storage Connector –

Seamlessly import from and export to Azure Blob Storage, expanding your data integration options.

#### Microsoft Dynamics 365 Connector –

Connect directly to Microsoft Dynamics 365 to sync and transform your business data.

#### Condition Builder UI Improvements –

The condition builder interface has been updated for a more intuitive and accessible experience.

#### File Collection Orchestration –

Automatically retrieve files from external cloud storage on a schedule and deposit them directly into Mammoth, eliminating manual downloads and uploads.

#### Notifications API v2 –

Notifications API now uses the latest v2 specification with improved reliability and extensibility.

#### JSON Column Header Menu –

Extract JSON column headers now appear in the context menu for direct access to column extraction.
