### Product Enhancement :

* Insights & Data Quality: Removed the Enabled checkbox from the Insights and Data Quality sections to simplify configuration and reduce unnecessary controls.

* Export Experience: Updated the Export experience to align with the standard Function Panel pattern, improving consistency, scalability, and ease of modification across pipeline tasks.

### Defect Fixes :
Resolved the following issues:

* Improved the visual feedback during file upload, ensuring clearer indication when files are dragged and dropped successfully.
* Fixed an issue in AI Prompt (Beta) where unnecessary details were displayed in the prompt modal.
* Resolved a validation issue in Generate Dataset via AI (Beta) where user-defined fields could be left empty.
* Fixed an issue in AI Prompt (Beta) where the Contains condition was not appearing correctly in rule details within the modal.
* Improved the Third-Party Connections experience by hiding the Schema option when only a single schema is available.
* Addressed UI inconsistencies during Parquet connection creation to improve overall usability.
* Fixed an issue in Group & Pivot where date components (such as month, year, weekday, and datetime) were not displayed correctly in the Rows and Columns sections.
