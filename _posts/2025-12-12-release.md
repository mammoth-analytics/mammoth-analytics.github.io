### Product Enhancement :

* **Data library – Add Data Experience -**
  * Introduced a redesigned Add Data modal with an improved layout to make data ingestion more intuitive and efficient.


* **Workspace Management-**
  * Added a new Leave Workspace flow, allowing users to leave a workspace from User Management and Profile Settings.
  * When the user is the sole workspace owner, a transfer ownership flow is triggered before leaving.

* **Improved Orchestration (Alpha feature)-**

  * Added a Run Now option on the Orchestration page to manually trigger orchestrations.
  * Applied UI improvements to enhance usability and clarity across orchestration workflows.

* Applied minor UI improvements to intent-based data transformations.


### Defect Fixes :
Resolved the following issues:

* Fixed an issue where the Preview panel did not display folder names correctly when only folders were present on the landing page.
* Corrected the Text Formatting rule description to ensure it displays accurately.
* Fixed an issue in Settings → Profile where changing the user name required a page refresh to reflect updates.
* Resolved an issue in Search & Replace where the Apply action was not enabled despite valid field and value selections.
* Fixed an issue in Bulk Replace where whitespace characters were not displayed correctly.
* Resolved an issue where double-clicking a column name did not enable the column rename action.
* Fixed an issue in Settings → User Management where removing a user from a workspace was not working as expected.
* Resolved an issue in Orchestration where deleting an orchestration from the Preview panel was not functioning.
* Fixed an issue where the search icon in Third-Party table search was not working.
* Fixed an issue in the Data Editor where downloading data as XLSX did not generate the file correctly.
