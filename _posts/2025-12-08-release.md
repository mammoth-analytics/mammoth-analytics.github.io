### Product Enhancement :

* **Data Editor – Transformations -**
  * Added a new AI Prompt option in the Data Editor, allowing users to create transformation tasks using natural language.
  * Moved all transformation actions into a single modal tools interface, giving users the flexibility to use either AI Prompt or existing transformation tools.

* **Data Editor – Insights & Data Quality -**
  * Introduced a new Insights panel in the Data Editor that highlights key observations and potential data issues to help users better understand their data.
  * Enhanced the Data Quality modal with improved visibility into scores, completeness, critical issues, and actionable recommendations.

* **Export & Publish Experience -**
  * Updated the Export menu UI, relocating it to the top-right for easier access and better alignment with primary actions.
  * Introduced a new Publish as Dashboard modal with guided suggestions, enabling users to quickly create dashboards or describe custom dashboard requirements.

* **Orchestration updates(Alpha feature) -**
  * Continued enhancements to the Orchestration experience, including UI updates to improve usability and clarity.
  * Migrated existing automation schedules to the Orchestration framework to support the updated automation experience.
  
 
* **Group & Pivot -**
  * Introduced a unified Group & Pivot function that combines Crosstab and Group-Aggregate capabilities into a single task, allowing both operations to be performed within the same view.
  * Enhanced the Group & Pivot function with column concatenation support, enabling text columns to be aggregated directly within the grouping configuration.
  * Removed the standalone Concatenation function, as its functionality is now available within the Group & Pivot feature.

* Upgraded the Vertical Navbar UI to improve navigation clarity and overall usability.


### Defect Fixes :
Resolved the following issues:

* Fixed an issue in Orchestration where datasets could not be selected after using the Clear All option during dataset refresh.
* Resolved an intermittent issue where multiple file uploads were failing.
* Fixed an issue in Orchestration Messaging where dataset details were not visible on the Orchestration page.
* Corrected the default decimal formatting for Group & Pivot aggregate values, ensuring aggregate results now display with two decimal places by default.
* Fixed an issue in Orchestration where the interval was not displayed when creating a yearly schedule.
* Resolved an issue where the data grid appeared greyed out while Explore cards were loading.
