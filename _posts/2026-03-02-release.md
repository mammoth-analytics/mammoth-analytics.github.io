### Product Enhancement :

#### Google Cloud Storage (Bucket) Data Import –

Introduced Google Cloud Storage bucket integration to enable seamless data import from GCS with secure configuration and streamlined ingestion workflow.

#### Data Export Enhancement – SharePoint Integration –

Introduced SharePoint as a new export destination under the Data Editor, enabling users to securely export datasets to external SharePoint locations with configurable settings.

#### New Workspace Creation –

Enabled users to create new workspaces directly from the sidebar with integrated plan selection and payment setup, simplifying workspace expansion and onboarding.

#### Subscription Management Enhancement –

Enabled self-serve management of subscription plans, add-ons, and billing settings, removing the need for sales intervention.

#### In-App Plan Enforcement & Upgrade Flow –

Implemented automatic plan limit enforcement with guided self-serve upgrade flows, replacing the previous “contact sales” approach for a more seamless user experience.

#### Premium Feature Restrictions –

Implemented structured restrictions for premium features and add-ons, including storage, seats, connectors, and advanced capabilities, with clear upgrade prompts, comparison visibility, and seamless trial and preview support.

### Defect Fixes :
Resolved the following issues:

* Duplicate vertical scrollbars were displayed in the Elasticsearch Functions Panel modal.
* Duplicate View functionality was not working as expected.
* “View Insights” feature in the Data Editor was not functioning properly.
* The 2FA enable button was shown even when already enabled; this has been corrected to reflect the active status and remove the redundant action.
