### Product Enhancement :

* Dashboard (Alpha) – Enabled non-blocking dashboard edits with selective updates, allowing changes without requiring a full page refresh.

* Project-level Settings – Added centralized project settings to define date formats and manage project access consistently across workflows.

* Date Format Detection – Improved handling of ambiguous date columns during file processing by prompting users to review and apply the correct date format across relevant columns.

* Bulk Replace – Added an option to visually reveal whitespaces to make bulk replacements more accurate and predictable.

### Defect Fixes :
Resolved the following issues:

* Fixed an issue where users were not informed about pending pipeline changes after applying a task from the AI prompt.
